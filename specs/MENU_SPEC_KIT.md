# 🎮 Painel de Controle: Spec Kit

Este arquivo serve como o seu menu interativo para os comandos do Spec Kit. Use-os no chat para guiar o assistente.

## 🚀 Fluxo de Trabalho (Workflow)

| Comando | Descrição | Quando usar? |
|---------|-----------|--------------|
| `/speckit.specify` | Cria a especificação | No início de uma nova funcionalidade. |
| `/speckit.plan` | Cria o plano técnico | Após a especificação ser aprovada. |
| `/speckit.tasks` | Gera lista de tarefas | Após o plano técnico estar pronto. |
| `/speckit.implement` | Inicia a programação | Quando estiver pronto para codificar as tarefas. |

---

## 🛠️ Utilitários de Documentação

- `/speckit.analyze` : Analisa o código atual para entender o contexto.
- `/speckit.checklist` : Valida se a especificação está completa.
- `/speckit.clarify` : Resolve dúvidas sobre a spec com perguntas ao usuário.

---

## 🐙 Comandos de Git

- `/speckit.git.feature` : Cria uma nova branch para a funcionalidade.
- `/speckit.git.commit` : Faz commit automático das mudanças do Spec Kit.
- `/speckit.git.validate` : Verifica se o repositório está em estado íntegro.

---

## 📘 Princípios (Constitution)

- `/speckit.constitution` : Configura as regras e tecnologias do projeto.
